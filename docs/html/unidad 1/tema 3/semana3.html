GESTIÓN EN LA TOMA DE DECISIONES

Elementos de un Problema de Decisiones

Un problema de decisiones implica la necesidad de elegir entre diferentes alternativas en un contexto específico. Los elementos clave de un problema de decisiones incluyen:

  • Contexto del Problema
  • Alternativas
  • Criterios y Preferencias
  • Incertidumbre y Riesgo
  • Resultados y Consecuencias
  • Toma de Decisiones
